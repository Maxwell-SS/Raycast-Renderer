# Raycast Renderer
My attempt at making a basic raycasting renderer inspired by Wolfenstein 3D, Made with c++ and sdl.

## building
```sh
git clone https://github.com/Maxwell-SS/Raycast-Renderer.git
cd Raycast-Renderer
mkdir build
cd build
cmake ..
make
./RaycastRenderer
```

## controls
- **W**: Move forward
- **S**: Move backward
- **A**: Turn left
- **D**: Turn right
